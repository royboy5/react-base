# react-base

Base project for React.

Built with,
- Webpack
- Webpack Dev Server
- Live Server
- Babel
- ESLint
- Prettier
- React
- React Router
- Tailwind CSS

## Setup
To install dependancies
```
$ npm i
```

## Weback Dev Server
```
$ npm start
```

## Build
Builds will be outputted to the `/dist` folder
```
$ npm run build
```

## Run production build
```
$ npm run serve
```
